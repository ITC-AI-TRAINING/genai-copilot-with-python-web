About JokeApp

JokeApp is a web application designed to deliver jokes from the JokeAPI, providing users with instant humor across various categories like programming, miscellaneous, and dark humor. Whether you're looking for a quick laugh or exploring specific joke types, JokeApp makes it easy and fun.

Technology Stack

This application is built using modern web technologies:

  • Flask: A lightweight Python web framework for backend logic and routing.
  • Bootstrap 5: For responsive, mobile-first UI components and styling.
  • JokeAPI: An external API providing a vast collection of jokes.

Features

  • Random Jokes: Get instant random jokes from various categories.
  • Category Selection: Choose from programming, miscellaneous, or dark humor.
  • Custom Filters: Apply filters to refine joke results.
  • Responsive Design: Works seamlessly on desktop and mobile devices.

Development with GitHub Copilot

This app was developed with the assistance of GitHub Copilot, an AI-powered coding assistant. Copilot helped generate code snippets, suggest improvements, and accelerate the development process, making it easier to build features like routing, templating, and UI components.
